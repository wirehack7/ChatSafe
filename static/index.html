ChatSafe

Ende-zu-Ende-verschlüsselte Chats — direkt im Browser

So nutzt du ChatSafe sicher

  1. Öffne ChatSafe im privaten/Inkognito-Modus.
  2. Gib deinem Chat einen Namen und klicke auf Chat erstellen.
  3. Sende die generierte URL an alle, die teilnehmen sollen – sie enthält den Verschlüsselungsschlüssel.
  4. Sobald alle beigetreten sind: Chat sperren, damit keine weiteren Personen eintreten können.
  5. Am Ende: entsperren, Nachrichten löschen und fertig.

So funktioniert die Verschlüsselung

Beim Erstellen wird ein kryptografisch sicherer Zufallsschlüssel generiert und im URL-Hash gespeichert – der Teil, den der Browser nicht ans Server überträgt. Alle Nachrichten werden per AES-256-GCM (Web Crypto API) im Browser ver- und entschlüsselt. Der Server sieht niemals den Klartext.

Warum ChatSafe?

Privatsphäre ist ein Grundrecht. ChatSafe ermöglicht schnelle, verschlüsselte Unterhaltungen ohne Account, ohne Tracking und ohne dass ein Dienst deine Nachrichten lesen kann – selbst wenn der Server kompromittiert ist.
